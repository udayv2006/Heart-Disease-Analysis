# ❤️ Heart Disease Analysis (Tableau Project)

## 📌 Project Overview
This project focuses on analyzing heart disease data using **Tableau** to uncover meaningful insights and visualize risk factors. The goal is to make healthcare data more accessible and actionable through interactive dashboards and storytelling.

---

## 🎯 Objectives
- Perform data analytics on heart disease datasets.
- Build interactive Tableau dashboards for visualization.
- Present findings through Tableau stories.
- Share a demo video for walkthrough and explanation.

---

## 📊 Features
- **Dashboard:** Interactive charts and graphs to explore patient data and risk factors.
- **Story:** A guided narrative highlighting key insights and trends.
- **Demo Video:** A walkthrough of the project explaining the analysis and visualizations.
